ProjectDataManager binary file format

Список проектов
projects.ini
{
	str: name
	str: path
}

Данные конкетного проекта
wdproj.ini
{
	// count: 6
	str: name
	str: path
	str: description
	str: author
	str: main html page
	str: wdstyle name
}

============== Файлы-ссылки ==============

web страница
*.wdweb
{
	byte: flag
	{
		html = 1
		md   = 2
		css  = 4
	}
}

глобальны фалй для css и html
*.wdstyle
{
	byte: flag
	{
		html = 1
		css  = 2
	}
}
